# 贪吃蛇游戏（Python实现）

## 📌 简介
这是我大一上学期Python大作业写的贪吃蛇游戏，支持键盘方向键控制，有得分显示和游戏结束判断。

## 🎮 玩法
- 方向键控制蛇的移动
- 吃到食物变长，得分+10
- 撞到自己游戏结束
- 按F1食物位置切换
- 按住空格得分增加999（这是一个小彩蛋）

## 🛠️ 运行环境
- Python 3.x
- 需要用到的库：pygame

## 📂 文件说明
- `snake.py`：主程序文件
